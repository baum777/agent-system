# Workflows

## Workflow A: New Feature / Refactor
1) Lead: update team_plan.md
2) Implementer: PR mit required PR sections
3) Reviewer: risk rating + required tests
4) QA: golden tasks + scorecard
5) Release captain: docs + changelog

## Workflow B: Hotfix
1) Lead: declare hotfix in team_progress.md
2) Implementer: minimal diff
3) Reviewer: mandatory approval
4) QA: smoke subset
