# Agententypen

1. **Knowledge Agent** – Recherche, Quellen, Vektorzugriff.
2. **Project Agent** – Workflow, Phasen, Projekt-Metadaten.
3. **Documentation Agent** – Produktion von Write-Only-Content mit Review.
4. **Junior Agent** – Ausführende Aktionen mit Senior-Review-Pflicht.
5. **Governance Agent** – Policy-Checks, Logs, Eskalationen.
