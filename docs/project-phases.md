# Projektphasen

- **Discovery** – Fragen klären, Hypothesen formulieren.
- **Design** – Lösung entwerfen, Risiken benennen.
- **Delivery** – Umsetzung und Dokumentation.
- **Consulting** – Kickoff, Analyse, Recommendation.

Alle Phasen werden über `packages/workflow` als YAML-Definitionen gehandhabt.
